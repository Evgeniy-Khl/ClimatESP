# История изменений проекта Climat_ESP8266

## [2026-06-03]
### Добавлено
- Фоновый механизм автоматического переподключения к Wi-Fi (`handleWiFi`). Попытки каждые 120 секунд.
- Динамический запуск Web-сервера и Telegram-бота при появлении сети.
- Регистрация маршрутов Web-сервера (`setupWebServerRoutes`) происходит сразу при загрузке.

### Исправлено
- **Критическое:** Устранена проблема сброса программы инкубации при ошибке `rtc.lostPower()`. Теперь программа восстанавливается из EEPROM даже при сбитом времени.
- Исправлено застревание при инициализации WiFiManager: теперь сервисы (NTP, Telegram) корректно инициализируются при любом успешном подключении (первичном или повторном).
- Оптимизирована логика `initWiFiManag`: разделены этапы настройки и запуска сервисов.
